### 〽️ Getting started

```zsh
    # Clonando o repositório em sua máquina
    $ git clone https://github.com/edsonjaguiar/crud-drinks

    # Acessando o repositório
    $ cd crud-drinks

    # Instalando as dependências
    $ yarn

    # Iniciando o server
    $ yarn dev
```
